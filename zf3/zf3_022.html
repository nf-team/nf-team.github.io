Открытое письмо И.Гульеву и издательству ДМК

    Копии: в электронные эхо-конференции relcom.talks,  relcom.  msdos,
relcom.os.windows,   fido.su.virus,   relcom.fido.ru.hacker;  издателям
электронных журналов MoonBug, Земский Фершал, LMD.

;     В начале 1999 г. московским издательством "ДМК" была издана тиражом
; 5000 экз. книга:
;
;    И.Гульев.
;    Компьютерные вирусы.Взгляд изнутри /
;    М.:ДМК, 1999.- 304 c.
;
;     Содержание большей  части  книги представляет собой почти дословную
; компиляцию  ряда  размещенных  в  сети  Интернет  материалов  различных
; авторов,  причем  эти  материалы были использованы без их уведомления и
; согласия.  Более того,  авторы просто не знакомы с И.Гульевым ни лично,
; ни заочно.
;     Многие из материалов этой книги носят весьма  сомнительный характер
; с  точки  зрения  статей  273  (создание  и распространение вредоносных
; программ) и 272 (неправомерный доступ к компьютерной информации) УК РФ.
; Относительно    немногочисленные    фрагменты   позитивного   характера
; воспроизведены с искажениями, снижающими их информативную ценность.
;     Мы, возмущенные   авторы  материалов  позитивной  направленности  и
; другие подписчики электронных конференций, фактом обнародования данного
; письма выступаем с критикой в адрес нечистоплотного "автора" и выражаем
; сожаление  по  поводу  публикации  уважаемым  московским  издательством
; материалов столь сомнительного свойства.
;     Мы требуем от плагиатора и издательства публичных извинений  в  тех
; же источниках информации, где будет опубликовано данное послание.
