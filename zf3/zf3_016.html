Антивирус PLY0D20
                              by Sassa

    Данный антивирус написан на третий этап конкурса антивирусов  и  не
является   ни   релизового,  ни  пререлизового  качества,  ни  качества
бета-версии, ни даже качества Микрософт-бета-релиза. J

    Он направлен на борьбу с единственным вирусом - PLY.3360  ( PLY0D20
).  Позволяет тестировать файлы,  лечить их, вакцинировать против этого
вируса.

    В текущем каталоге обязательно присутствие файла  сигнатуры вируса:
PLY0D20.SIG.  При  его  отсутствии  будет  выжано сообщение и антивирус
прекратит работу.

    Любые существующие файлы с именем PLY0D20.TMP  в  текущем  каталоге
будут   уничтожены,  т.к.  используются  антивирусом  при  лечении  для
временного  хранения  данных.  Создавать  такой  файл  _специально_  не
обязательно.


При запуске антивируса обязательно указывать параметр
   < путь, маска >,
    причем путь - не  обязательно,  по  умолчанию  принимается  текущий
каталог на текущем диске. Маска указывает файлы для тестирования:
 *.*
 *.EXE
или любая другая, разрешенная DOS.

    Запуск без параметров приведет к тому, что просто отобразится экран
помощи.

Возможно использование дополнительных параметров - ключей:
-[ A C V N ].
  -A  - спрашивать проверку каталога: будет спрашивать разрешения
проверить подкаталог. Без указания этого параметра проверяет указанный
каталог с подкаталогами автоматически.

    -C - лечить молча. Не будет ни спрашивать, ни отображать информацию
о зараженности файлов.  Без указания этого параметра  просто  тестирует
файлы и при обнаружении зараженного выдает запрос на лечение.

    -V -    вакцинировать    автоматически.   Все   EXE   файлы   будут
вакцинироваться  автоматически.  Без  указания  этого  параметра  будет
спрашивать разрешение на вакцинирование каждого EXE файла.

    Вакцинация -   замена   ChkSum   в  MZ-Headerе  на  2020h  (  метка
зараженности вирусом ).  Вирус думает, что заражено, хотя на самом деле
нет.  Можно  вакцинировать файлы без использования данного антивируса -
просто переименовать EXE файлы в COM файлы,  но  существуют  программы,
очень любящие свое имя, а особенно расширение. K

    -N -  не  вакцинировать  ни  при  каких  обстоятельствах.  Не будет
задавать глупых вопросов ( но не будет и вакцинировать J

    Если Вам надоело ждать конца тестирования, надавите лучше ESC, а не
RESET J.

    ESC, нажатое  во время вопроса распознается как НЕТ( не лечить,  не
вакцинировать ).
